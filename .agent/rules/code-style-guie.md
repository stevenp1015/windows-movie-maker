# **Code Aesthetics**

**Ensure that your code itself is:**

- Structurally organized in a _thoughtful_ way, not just random mindless structure
- Aesthetically pleasing methods of formatting, spacing, separation, line breaks, etc..
- Impress me more than how impressed I am when I format with Prettier or Beautifier; You Can Do Better Than Them.
